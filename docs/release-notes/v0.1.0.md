# Local Context Bridge v0.1.0

First public desktop release of Local Context Bridge.

Local Context Bridge is a Windows-first local desktop tool for selectively moving TRAE SOLO CN conversation context between accounts on the same machine. It is intended for local data portability: discover the app and database, inspect non-deleted conversations, preview a transfer plan, back up the live database, and apply the selected project or session transfer to the current local account.

## Highlights

- Auto-discovers TRAE SOLO CN app, local database, logs, workspace, and backup locations.
- Lets users manually browse for the app executable or `database.db` when discovery fails.
- Detects all database user IDs and current-user candidates from local logs.
- Supports project-level and session-level transfer previews.
- Excludes deleted sessions by default.
- Backs up live database files before writeback.
- Builds as a GUI desktop app, so the installed app does not open a console window.

## Safety Notes

- This build is unsigned; Windows SmartScreen may show a warning.
- Close TRAE SOLO CN before applying a transfer.
- Keep the generated backup until you have verified the restored conversations.
- Use this only with local data that you are allowed to access.
