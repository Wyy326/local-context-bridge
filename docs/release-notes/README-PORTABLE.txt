Local Context Bridge v0.1.2 Portable Package

This ZIP is the complete portable desktop package for Windows users who do not want an installer.

Contents:

- Local Context Bridge.exe
- README.md
- LICENSE
- RELEASE_NOTES.md
- README-PORTABLE.txt

How to run:

1. Extract the ZIP to a normal user-writable folder.
2. Run "Local Context Bridge.exe".
3. If Windows SmartScreen appears, review the warning and continue only if you trust this build.

Notes:

- WebView2 Runtime is required. Most Windows 10/11 systems already include it.
- The app works with local files only and does not upload databases, logs, account IDs, or telemetry.
- Close TRAE SOLO CN before applying a transfer.
- Keep the generated backup until you have verified that the restored conversations are visible.
